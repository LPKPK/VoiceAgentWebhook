# Retell AI Webhook Test

## Overview

Here are the steps I have been through to learn and make my first voice agent work!

## Prerequisites

- FastAPI
- Uvicorn
- Retell Python SDK
- Postman (for local testing)
- ngrok (for exposing the local server)

Set up environment variable

```bash
conda env config vars set RETELL_API_KEY="my_retell_api_key"
```
## Running the Server

Start the FastAPI application:

```bash
uvicorn voice_agent:app --reload
```

Testing locally using Postman, following the setup guide: <https://docs.retellai.com/features/register-webhook>

Then push it online, expose the server using ngrok:
```bash
ngrok http http://127.0.0.1:8000
```

Copy the HTTPS URL generated by ngrok and configure it as the webhook URL in the Retell dashboard.

## Result

The FastAPI webhook successfully receives webhook events from Retell AI and processes the supported event types correctly.
